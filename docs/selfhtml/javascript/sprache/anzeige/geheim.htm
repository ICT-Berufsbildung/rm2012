Hallo Traumtänzer!

Diese Art von Passwortschutz ist allerdings nicht besonders ernst zu nehmen. Sie wollen wissen, wie man wirklich Web-Seiten mit Passwörtern schützen kann? Zuverlässiger Passwortschutz ist nur server-seitig lösbar. Die im Web übliche Technik dafür ist die so genannte .htaccess-Datei.

In SELFHTML wird ein solcher Passwortschutz auf der Seite  .htaccess - Server-Reaktionen kontrollieren beschrieben.
Im Online-Angebot von SELFHTML aktuell finden Sie zudem folgende Artikel:
 htaccess-FAQ
 htaccess Zugriffskontrolle

Fragen Sie gegebenenfalls Ihren Provider, ob er Verzeichnisschutz via htaccess anbietet!
