CSS-Browserweichen

Die folgenden Beispiele werden standardkonform mit grüner Schrift im Rahmen angezeigt, ansonsten mit roter Schrift ohne Rahmen.

Attribut-Selektor: wird vom Internet Explorer bis zur Version 6 ignoriert und definiert hier alle Rahmen.

Kind-Selektor: wird vom Internet Explorer bis zur Version 6 ignoriert.

Star-HTML: wird fälschlich vom Internet Explorer bis zur Version 6 interpretiert.

Star-Plus-HTML: wird fälschlich vom Internet Explorer 7 interpretiert.

Selektor-Kommentar: wird vom Internet Explorer bis Version 5 für Windows und Mac ignoriert.

Medienangabe: wird vom Internet Explorer 4 für Windows und den Mac-Versionen ignoriert.

Element-spezifischer Selektor: wird vom Netscape 4 ignoriert.

Tantek-Hack: wird vom Internet Explorer erst ab Version 6 korrekt interpretiert
- im Netscape 4 kommt es hierdurch zu Fehlern im gesamten Style-Bereich!

Tan-Hack: beeinflusst nur den Internet Explorer vor Version 6
- eine Kombination des Star-HTML- und des Tantek-Hacks für einzelne Definitionen.
