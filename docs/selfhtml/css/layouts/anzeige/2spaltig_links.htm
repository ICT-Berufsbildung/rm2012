Navigation links gefloatet

In diese Seite sind nun auch die CSS-Definitionen zur Präsentation eingebunden, durch die u.a. die Navigation links neben dem Inhaltsbereich platziert wird.

Dennoch wäre diese Seite auch ohne CSS nutzbar und würde dann vom Browser entsprechend dessen Voreinstellungen auch sinnvoll angezeigt werden.

Im Internet Explorer bis zur Version 6 werden Sie an dieser Stelle vermutlich auch einen Bug erkennen, der den neben einem floatenden Element stehenden Text zusätzlich um 3px einrückt.
