Der Absatz, den Sie gerade lesen, wurde mit text-align als Blocksatz ausgerichtet. Ferner wurde ein Rand von 50px eingestellt, just for fun oder auch, damit Sie sehen, wie die einzelnen Stylesheet-Angaben zusammenspielen. Lange hat es gedauert, bis die ersten Browser in der Lage waren, Blocksatz darzustellen. Mit den 4er-Versionen der beiden führenden Produkte von Netscape und Microsoft ist das jedoch problemlos möglich. Damit der Blocksatz auch wirklich sichtbar wird, enthält dieser Textabsatz diesen überflüssigen Text, der sich aber hoffentlich wenigstens flüssig liest.
