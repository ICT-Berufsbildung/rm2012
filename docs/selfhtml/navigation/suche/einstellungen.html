Maximale Trefferzahl pro Seite

Stellen Sie hier ein, wieviele Treffer gleichzeitig pro Seite angezeigt werden sollen. Bedenken Sie, dass einzelne Suchen deutlich länger dauern können, wenn Sie hier "alle" wählen. Der Vorgabewert ist 50.

Ausgabeformat

Hier können Sie einstellen, ob bei der "Wortanfang"- und "Oder"-Suche auch die in den Dokumenten vorkommenden zutreffenden Suchbegriffe angezeigt werden sollen (detailliert) oder nicht (einfach).

detailliert
einfach

Gefundene Dateien öffnen...

Hier können Sie einstellen, wo sich die Dokumente öffnen, wenn Sie in der Ergebnisliste auf die Überschrift klicken.

im selben Frame
in einem zweiten Fenster
jedes in einem eigenen Fenster

Hervorhebung der Treffer

Wenn Sie ein gefundenes Dokument geöffnet haben, können Sie darin über die Schaltfläche "Treffer markieren" im oberen Frame die Suchbegriffe farbig hervorheben lassen. Sie können hier einstellen, ob für alle Begriffe dieselbe Farbe verwendet werden soll oder ob mehrere Farben verwendet werden sollen.

gelbe Hervorhebung 
mehrfarbige Hervorhebung
