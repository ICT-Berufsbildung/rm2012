Funktionsweise

Die vorliegende Suchfunktion ermöglicht eine komfortable Volltext-Suche über das gesamte Dokument. Sie ist vollständig mittels  JavaScript realisiert.

Die Suchmaschine ist in erster Linie für den lokalen Einsatz konzipiert, also für den Fall, dass Sie SELFHTML von der Festplatte oder von einer CD nutzen. Seit der Version für SELFHTML 8.1 kann sie aber auch direkt im Internet genutzt werden. Auch in diesem Fall erfolgen die Berechnungen für die Ermittlung des Suchergebnisses nicht auf dem  Server, auf dem SELFHTML liegt, sondern - dezentral - auf den Rechnern der Nutzer.

Wenn Sie SELFHTML im Internet lesen, haben Sie außerdem die Möglichkeit, die  Online-Suche zu verwenden.

So formulieren Sie Suchanfragen

Folgende Suchmöglichkeiten stehen zur Verfügung:

Einfache Suche:

Geben Sie einfach einen oder mehrere Suchbegriffe in das obige Suchfeld ein.
Gefunden werden alle Dokumente, in denen die Suchbegriffe vorkommen (UND-Verknüpfung: alle Begriffe müssen vorkommen; ODER-Verknüpfung: mindestens einer der Begriffe muss vorkommen; geben Sie "und" bzw. "oder" nicht im Suchfeld ein, sondern benutzen Sie zur Auswahl die Schaltflächen). Zwischen Groß- und Kleinschreibung wird nicht unterschieden.

Sie können wählen zwischen

  • Suche nach ganzen Wörtern
  • Suche nach Wortanfängen

Die Auswahl erfolgt mittels der obigen Schaltflächen und wird automatisch auf alle Suchbegriffe angewandt.

Außerdem können Sie bestimmen, dass Seiten, in denen ein bestimmter oder bestimmte Begriffe vorkommen, nicht angezeigt werden. Für diese Ohne-Suche geben Sie ein "-" vor dem Begriff ein.

Phrasensuche:

Geben Sie zwei oder mehr Wörter innerhalb von Gänsefüßchen ein.
Bsp: "zeichen maskieren"
Gefunden werden Textstellen, in denen die angegebenen Wörter genau hintereinander vorkommen.

Diese Art der Suche lässt sich mit der Suche nach Wortanfängen kombinieren:
Wenn Sie mit dieser Option z.B eingeben: "regulär ausdr", werden "regulärer Ausdruck", "reguläre Ausdrücke" usw. gefunden.

Satzzeichen wie Punkt oder Komma werden automatisch ignoriert.

Weitere Funktionen:

Aus der Ergebnisliste können Sie die gefundenen Dokumente direkt zur Ansicht auswählen. Über die Option "Suchleiste schließen" können Sie dann die Frameansicht beenden und das Dokument ins Vollbild nehmen.

Wenn Sie die Suchleiste beibehalten, können Sie nicht nur jederzeit eine neue Suchanfrage formulieren, sondern auch über den Schalter "Treffer markieren" den oder die Suchtreffer im Dokument farbig hervorheben. Das setzt allerdings voraus, dass Sie nicht, wie es mittels der  Einstellungen möglich ist, die Dokumente in einem neuen Fenster öffnen lassen.

Mit der Tastenkombination [ALT] + u (für Suchen) können Sie das Eingabefeld der Suchleiste aktivieren. Sie können auch Wörter im Dokument mit der Maus markieren und nach oben in das Eingabefeld ziehen, um mit ihnen eine neue Suche durchzuführen.

Kontakt

Bitte richten Sie alle Fragen, Vorschläge, Fehlermeldungen usw. zu dieser Suchmaschine nicht an die SELFHTML-Redaktion, sondern direkt an den Autor der Suchmaschine:  Oliver García, selfsuche@ogu.de.  Eventuelle neue Versionen der Suchmaschine für Selfhtml 8.x werden vom Autor im Internet zur Verfügung gestellt.

Unter dieser Adresse können Sie sich auch informieren, falls Sie sich für die Lizenzierung einer angepassten Version dieser Suchmaschine für Ihre eigenen Projekte, die z.B. auf CD-ROM veröffentlicht werden sollen, interessieren.
