Verwendete Symbole

Die folgenden Symbole werden in SELFHTML verwendet.

Symbol Bedeutung
Geschlossener kleiner Ordner - wenn Sie den daneben stehenden Verweis anklicken, gelangen Sie zur Übersichtsseite eines größeren Abschnitts innerhalb von SELFHTML.
Geöffneter kleiner Ordner - wenn Sie den daneben stehenden Verweis anklicken, gelangen Sie zu einem übergeordneten Bereich von SELFHTML. Verweise dieser Art finden Sie in der Kopf- und Fußleiste auf den einzelnen Seiten. Mehrere übergeordnete Ebenen werden von links an eingerückt dargestellt.
Geschlossener großer Ordner - Dieses Symbol kommt im hellgrauen Kopfbereich von Seiten vor und bedeutet: die aktuelle Seite ist eine Kapitelübersichtsseite. Die Seite enthält selbst keine besonderen Informationen, sondern lediglich Verweise zu untergeordneten Seiten, in denen die eigentlichen Informationen enthalten sind.
Kleines Seitensymbol. Datei oder Textstelle innerhalb des Dokuments - wenn Sie den daneben stehenden Verweis anklicken, gelangen Sie zu einer Stelle mit konkreter Information. Das Symbol wird auch bei Querverweisen verwendet.
Kleines Seitensymbol für Beispiel-Seiten, z.B. Seiten mit Anzeigebeispielen oder Quellenhinweisen. Wenn Sie den daneben stehenden Verweis anklicken, wird die entsprechende Seite geladen.
Großes Seitensymbol - Dieses Symbol kommt im hellgrauen Kopfbereich von Seiten vor und bedeutet: die aktuelle Seite ist eine informationstragende Seite.
Pfeilsymbol für eine Textstelle auf der gleichen Seite weiter unten - wenn Sie den daneben stehenden Verweis anklicken, gelangen Sie zu einer Stelle innerhalb der gleichen Datei weiter unten.
In hellgrauen Balken ohne nebenstehenden Verweis: anklicken, um zum Seitenende zu springen.
Pfeilsymbol für eine Textstelle auf der gleichen Seite weiter oben - wenn Sie den daneben stehenden Verweis anklicken, gelangen Sie zu einer Stelle innerhalb der gleichen Datei weiter oben.
In hellgrauen Balken ohne nebenstehenden Verweis: anklicken, um zum Seitenanfang zu springen.
Pfeilsymbol für die vorhergehende Seite. Steht am Ende von Informationsseiten und bedeutet: zur vorhergehenden Seite blättern.
Pfeilsymbol für die nächste Seite. Steht am Ende von Informationsseiten und bedeutet: zur nächsten Seite blättern.
Symbol für E-Mail. Wenn Sie den daneben stehenden Verweis anklicken, können Sie eine E-Mail an die betreffende Person absetzen.
Geschlossener kleiner Ordner; hautfarben - wenn Sie den daneben stehenden Verweis anklicken, gelangen Sie zu einer Übersichtsseite eines größeren Abschnitts im Online-Angebot von SELFHTML aktuell oder im Online-Angebot eines Partnerprojekts von SELFHTML. Das Verweisziel hat das gleiche Layout wie die Seiten von SELFHTML. Um solche Verweise ausführen zu können, ist eine Internet-Verbindung erforderlich.
Geschlossenes kleines Seitensymbol; hautfarben - wenn Sie den daneben stehenden Verweis anklicken, gelangen Sie zu einer Informationsseite im Online-Angebot von SELFHTML aktuell oder im Online-Angebot eines Partnerprojekts von SELFHTML. Das Verweisziel hat das gleiche Layout wie die Seiten von SELFHTML. Um solche Verweise ausführen zu können, ist eine Internet-Verbindung erforderlich.

Symbol für englischsprachige Seite. Steht vor Verweisen zu externen, englischsprachigen Web-Angeboten (wird also als Sprachensymbol verwendet, nicht als Ländersymbol).

Tipps und Tricks zum Arbeiten mit SELFHTML

Die folgenden Hinweise sollen Ihnen helfen, SELFHTML optimal zu nutzen.

  • Falls Sie das Dokument gerade online lesen und für Ihre Online-Verbindung Zeit- oder Volumengebühren zahlen: laden Sie sich das Dokument besser auf Ihren Rechner. Sie können dieses Dokument herunterladen (siehe  SELFHTML im Web). So können Sie das Dokument ohne Stress lesen.
  • Kopieren Sie sich Beispiele zu Quelltexten, die Sie verwenden wollen, einfach über die Datenaustausch-Schnittstelle (Zwischenablage) Ihres Rechners direkt aus diesem Dokument in Ihren Editor. Dann brauchen Sie den Code nur noch anzupassen und vermeiden entscheidende Tippfehler. Dies gilt jedoch nur, wenn Sie code-basiert, also mit einem Texteditor oder einem textorientierten HTML-Editor arbeiten, nicht bei Verwendung eines grafischen WYSIWYG-Editors.
  • Machen Sie sich mit den verschiedenen  Navigationshilfen vertraut, um schneller mit SELFHTML arbeiten zu können und schneller an gewünschte Information heranzukommen.
