Begriff Definition
Avatar Bezeichnet in der Mythologie einen Gott, der auf Erden wandelt. Im Online-Sprachgebrauch eine grafische Phantasiefigur, mit der ein Teilnehmer an Chats, Rollenspielen usw. teilnimmt.
Award Im WWW eine Auszeichnung für besonders gut gemachte Web-Seiten. Eine Inflation von selbsternannten Award-Verleihern, deren Awards nur den Zweck haben, Besucher auf die eigenen Seiten zu locken, haben das Award-Wesen im Web jedoch stark beschädigt.
