XML/DTDs

Sinnvolle Vorkenntnisse zum Verständnis:  HTML,  Web-Technologien allgemein und speziell die einführenden Bemerkungen zu  XML.
Dieser Abschnitt behandelt den Aufbau von XML-Dateien, das Definieren eigener Auszeichnungssprachen mit Hilfe von XML-DTDs und die Möglichkeiten der Präsentation von XML-Daten.
Der Abschnitt besteht aus folgenden Seiten und Unterabschnitten:

 Einführung in XML
 Regeln für XML-Dateien
 Dokumenttyp-Definitionen (DTDs)
 Darstellung von XML-Daten
