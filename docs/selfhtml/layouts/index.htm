Fertige Layouts

Dieser Abschnitt richtet sich an alle, die beim Erstellen einer eigenen Homepage lieber auf einem vorhandenen Layout aufsetzen und dieses nach ihrem Willen verändern, erweitern und ausgestalten wollen. Es werden verschiedene typische, aber individuelle Layouts für Web-Seiten vorgestellt. Die vorgestellten Layouts können Sie für eigene Web-Seiten als Vorlage und veränderbares Rohmaterial frei verwenden. Zu jedem Layout wird kurz beschrieben, welche Besonderheiten dabei zum Einsatz kommen, und welche Dateien dazugehören. Alle Layouts stammen von namhaften Web-Designern. Die Layouts benutzen HTML 4.0 und Stylesheets 1.0. Mit Browsern ab der 4er-Generation sollte es keine Probleme geben bei der Anzeige. Deutlich ältere Browser wie Netscape 3.x können einige Layouts wegen der fehlenden Unterstützung von CSS nicht richtig oder unschön darstellen.
